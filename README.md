## 🌾 AI-Powered Crop Yield Prediction System
📌 Overview

This project is an AI-based web application designed to predict crop yield using historical agricultural data and real-time weather information. It helps farmers and stakeholders make better decisions by providing accurate yield predictions and smart agricultural insights.

---

## 🚀 Features

🌱 Crop yield prediction using Machine Learning (XGBoost)
🌦️ Live weather data integration
🤖 AI-powered agriculture assistant
📊 Downloadable crop yield reports
📈 Data-driven insights for better farming decisions
💻 User-friendly web interface

---

## 🧠 Technologies Used

Python
Machine Learning (XGBoost)
Flask (Backend)
HTML, CSS, JavaScript (Frontend)
MySQL / SQLite (Database)
Weather API Integration

## ⚙️ Installation & Setup

Clone the repository:
git clone https://github.com/your-username/agri-yield-ai.git

---

## Navigate to the project folder:
cd agri-yield-ai

Install dependencies:
pip install -r requirements.txt

Run the application:
python app.py

Open in browser:
http://127.0.0.1:5000/

---

## 📊 How It Works

The model is trained using past agricultural data.
User inputs details like crop type, location, and environmental factors.
The system uses the trained XGBoost model to predict yield.
Live weather data enhances prediction accuracy.
Results are displayed and can be downloaded as a report
